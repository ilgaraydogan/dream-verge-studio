Dream Verge Studio is an open-source, native macOS application built for mental health professionals to analyze patient dreams using AI. It runs multiple AI models simultaneously, maintains patient soul profiles, flags suspicious patterns, and keeps everything organized in project-based workspaces — like Xcode, but for the unconscious mind.

---

## Why Dream Verge Studio?

Existing tools force professionals to copy-paste dream descriptions into generic AI chatbots, losing clinical context, session history, and structure. Dream Verge Studio is purpose-built for professional dream work:

- **Project-based workspace** — each patient is a project, each dream is a structured file
- **Multi-model simultaneous analysis** — run GPT-4o, Claude, Gemini, and more in parallel, compare outputs side by side
- **soul.md** — a living psyche profile per patient, auto-generated by AI or manually edited by the clinician
- **Suspicion flagging** — AI automatically highlights patterns that warrant clinical attention across sessions
- **Config-driven** — `.dreamconfig` file tunes analysis behavior per patient
- **100% local project files** — no cloud sync, no third-party storage of patient data
- **Your API keys** — BYOK (Bring Your Own Key), we never touch your data
- **Open source, MIT licensed** — audit it, fork it, build on it

## File Formats

### `.dream` — Dream Entry File
```
---
id: dream-2025-03-16-001
date: 2025-03-16
session: 4
tags: [water, chase, unknown-figure]
flagged: false
models_used: [gpt-4o, claude-3-5-sonnet]
---

Dream content goes here. Free text, written by the clinician
immediately after the patient's account.
```

### `soul.md` — Patient Soul Profile
```markdown
# Patient Soul Profile

## Core Themes
- Recurring: water, darkness, pursuit

## Significant Symbols
- Water → emotional overwhelm (confirmed session 2)
- Unknown figure → unresolved authority conflict (hypothesis)

## Flagged Patterns
- Session 3 & 4: escalating aggression themes — monitor

## Analyst Notes
...
```

### `.dreamconfig` — Analysis Configuration
```json
{
  "models": ["gpt-4o", "claude-3-5-sonnet"],
  "parallelAnalysis": true,
  "language": "tr",
  "flagging": {
    "enabled": true,
    "sensitivityLevel": "clinical",
    "keywords": ["violence", "death", "isolation"]
  },
  "soul": {
    "autoUpdate": true,
    "updateAfterSessions": 2
  }
}
```

## Getting Started (Development)

### Prerequisites
- macOS 13+
- Node.js 18+
- Rust (latest stable)
- Xcode Command Line Tools

### Setup
```bash
git clone https://github.com/ilgaraydogan/dream-verge-studio
cd dream-verge-studio
npm install
npm run tauri dev
```

### Build
```bash
npm run tauri build
```
